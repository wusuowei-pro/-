# 单数字图像识别系统 - MATLAB 模板匹配实现

## 项目功能
本项目是基于 MATLAB 开发的单印刷体数字图像识别系统，核心功能如下：
- 完成输入图像的全流程预处理：灰度化、3×3 中值滤波降噪、Otsu 自适应二值化、形态学面积去噪，可自动判断背景明暗并取反，统一为白字黑底格式。
- 通过连通域分析定位、提取图像中的数字目标区域，支持数字轮廓的可视化绘制。
- 自动生成 0-9 标准印刷体数字模板库，统一归一化为 28×28 像素规格。
- 基于二维皮尔逊相关系数实现模板匹配，选取相关性最高的模板输出最终识别结果。
- 提供全流程可视化展示，同步呈现原图、降噪图、二值图、轮廓图、目标区域、匹配得分、最佳模板与识别结果。

## 实验环境
- 开发平台：MATLAB R2016a

## 运行步骤
1. **图像读取与灰度化**：通过文件对话框选择输入图像，将彩色图像转换为单通道灰度图，降低计算量。
2. **降噪处理**：采用 3×3 中值滤波去除图像中的椒盐噪声与孤立噪点，同时保留数字边缘信息。
3. **自适应二值化**：使用 Otsu 算法自动计算最优分割阈值生成二值图；通过像素均值判断背景颜色，若背景为白色则自动取反，统一为白字黑底格式。
4. **形态学去噪**：移除面积小于 30 像素的连通域，清除残留的微小噪点，净化目标区域。
5. **目标数字提取**：通过 `regionprops` 提取所有连通域属性，选取面积最大的连通域作为待识别数字，裁剪得到数字区域的二值图。
6. **模板库生成**：在后台窗口循环渲染 0-9 标准数字，经二值化、裁剪空白、尺寸归一化后，生成 28×28 像素的标准模板库。
7. **模板匹配识别**：将待识别数字缩放至 28×28 像素，逐一计算与 10 个模板的二维相关系数，相关系数最大值对应的数字即为识别结果。
8. **结果可视化与输出**：以 2×4 子图布局展示全流程处理效果，并在控制台打印最终识别的数字。

## 实验结论
### 识别效果
- 对清晰、无严重倾斜的规范印刷体数字识别准确率较高。
- 中值滤波搭配形态学滤波具备良好的抗噪能力，Otsu 阈值可适配不同光照、对比度的输入图像。
- 单张图像处理耗时约 0.5~1 秒，运行速度符合轻量级场景需求。

### 算法优势
- 方案轻量化，无需深度学习训练。
- 流程模块化，各环节独立便于调试。
- 部署门槛低，可迁移至其他图像处理平台。
- 全流程可视化完善，适合算法原理演示与入门学习。

### 算法局限
- 高度依赖标准模板库，对手写数字、几何畸变（倾斜、笔画粗细变化）、多数字粘连、重度模糊/遮挡的场景识别能力差，泛化能力有限。
- 后续可通过扩充多模板库、引入倾斜校正、升级特征提取方式、结合机器学习分类器等方向优化提升。
